MDX est le format d’article par défaut de Lisible et le format de toutes les pages de cette documentation. Il conserve Markdown et permet d’importer des composants dans la prose. Réservez les interactions aux cas qui améliorent réellement la compréhension ; un paragraphe ou un tableau restent préférables pour une information statique.

Le composant expose un vrai `tablist`, supporte les flèches, <kbd>Home</kbd> et <kbd>End</kbd>, et ne charge React que sur les pages qui l’utilisent.

## Spoiler

Le composant masque une réponse jusqu’à une activation explicite : <Spoiler>le contenu reste présent dans le HTML et doit donc rester non sensible.</Spoiler>

:::warning[Pas pour des secrets]
Un spoiler est un effet de présentation, pas un contrôle d’accès. N’y placez jamais une clé, une réponse privée ou une donnée confidentielle.
:::

## Règles d’hydratation

- `client:load` pour une interaction immédiatement nécessaire ;
- `client:idle` pour une amélioration non critique ;
- `client:visible` pour un composant éloigné sous la ligne de flottaison ;
- aucun directive client pour un composant Astro statique.
